Dokumentation av det här APIet

Routes

  • GET /users - retunerar alla användare. Behöver en bearer token i authorization header
  • GET /users/{id} - retunerar en användare med ett angivet id. Behöver en bearer token i authorization header
  • GET /me - retunerar din information från bearer token. Behöver en bearer token i authorization header
  • POST /users - skapar en ny användare med hashat lösenord, accepterar JSON objekt på formatet {"firstname": "", "lastname": "", "userId": "", "passwd": ""}. Behöver en bearer token i authorization header
  • POST /login - loggar in på existerande användare. Accepterar JSON-objekt på formatet {"userId": "", "passwd": ""}. Retunerar en JWT Token som varar i 2h
